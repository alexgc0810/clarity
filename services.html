Servicios

Volumetría Hepática

Se obtiene imágenes tridimensionales del hígado, su sistema vascular y tumores o lesiones focales. Se pueden obtener volúmenes, y aplicar diversos planeamientos quirúrgicos. Se realiza a partir de imágenes de TC, pudiendo mezclar varias fases de contraste.

Precio: $100 dolares

Volumetría Renal

Se obtiene imágenes tridimensionales los riñones, su relación con estructuras óseas, así como imágenes del sistema vascular y excretor (ureteros). Se puede identificar masas o tumores, cálculos y quistes. Se pueden obtener volúmenes, y aplicar diversos planeamientos quirúrgicos que mejora la toma de decisiones. Se puede realizar simulación de laparoscopía, una vez que el cirujano establezca el posible abordaje. Se realiza a partir de imágenes de TC, pudiendo mezclar varias fases de contraste.

Precio: $100 dolares

Planeamiento Quirúrgico

En hígado, riñón y pulmón. Uso de software basado en Inteligencia Artificial, para reconocimiento de estructuras anatómicas, poder diferenciar diferentes lesiones y realizar imágenes 3D pudiendo realizar diversas simulaciones de cirugía.

Precio: $100 dolares

Análisis Pulmonar

Postprocesado avanzado basado en TC de tórax, se localiza el Nódulo, pudiendo ubicarlo con respecto a estructuras anatómicas como las cisuras o costillas; se segmenta el pulmón y se puede realizar reconstrucción de la vía aérea bronquial (broncoscopía virtual) para trazar el camino más cercano a la lesión. Se realiza análisis por histograma del nódulo. Se puede calcular volúmenes de los segmentos pulmonares. Realizar simulaciones quirúrgicas. Además se realiza medición de enfisema mediante indice de Goddard.

Precio: $100 dolares
Cuantificación de enfisema: $50 dolares

Seguimiento Oncológico

Análisis comparativo de estudios en pacientes oncológicos para medir respuesta al tratamiento. Se utiliza los indices internacionales (RECIST), siendo la ventaja de este método que el software que se utiliza puede realizar localización automática de las lesiones previas, cuando se cuenta con el estudio base, compaginando los nuevos estudios pese a que sean realizados en distintos equipos (registro no rígido asistido por inteligencia artificial). Posteriormente se pueden ir agregando nuevos estudios. Los resultados se pueden mostrar en imágenes 3D, y con un reporte del resultado de RECIST que cuantifica los cambios.

Precio: $100 dolares

PostProcesado Vascular

Contamos con diversas herramientas para segmentación de estructuras vasculares, en vasos del cuello (carótidas y vertebrales), aorta y sus ramas. Se pueden realizar mediciones precisas de estenosis o aneurismas, caracterizar placas, y planeamiento endovascular.

Precio: $100 dolares
