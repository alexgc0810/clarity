VOLUMETRIA RENAL

Consiste en una avanzada técnica de manipulación para el post procesado de imágenes mediante segmentación asistida con Inteligencia Artificial, utilizando el software Synapse 3D. Se obtiene imágenes tridimensionales los riñones, su relación con estructuras óseas, así como imágenes del sistema vascular y excretor (ureteros). Se puede identificar masas o tumores, cálculos y quistes. Se pueden obtener volúmenes, y aplicar diversos planeamientos quirúrgicos que mejora la toma de decisiones. Se puede realizar simulación de endoscopía, una vez que el cirujano establezca el posible abordaje. Se realiza a partir de imágenes de TC, pudiendo mezclar varias fases de contraste.
